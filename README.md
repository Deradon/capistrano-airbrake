# Capistrano::Airbrake

Airbrake integration for Capistrano

## Installation

Add this line to your application's Gemfile:

    gem 'capistrano', '~> 3.0'
    gem 'capistrano-airbrake'

And then execute:

    $ bundle

## Usage

Require in Capfile to use the default task:

    # Capfile
    require 'capistrano/rvm'

And you should be good to go!
